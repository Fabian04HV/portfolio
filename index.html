About Me

Welcome to my portfolio page! I am Fabian Hurtado Volkmann (*2004), an enthusiastic Fullstack Web Developer from Berlin.

After completing my high school education, I further improved my skills by attending the Fullstack Web Development Bootcamp at Ironhack School. Currently, I am studying "Business Administration - Digital Economy" at the Berliner Hochschule für Technik and I am looking for a student job as a Web Developer.

Infinite Recall

A Flashcard Learning App for Students

This MERN (MongoDB, Express.js, React.js, Node.js) application is the first complex app I have ever created all by myself. It was the final project of my Ironhack Web Development Bootcamp which I completed in May 2023.

I started with the user authentication (signup, login, logout) using jsonwebtoken. Authenticated users can create, edit, delete and learn with flashcard collections. You can also search for other student's flashcards and learn with them.

There are different learn modes that optimize your learning experience. You can also analyse your past learn sessions using the statistics page, where all your progress is tracked with a detailed summary and hints where you can improve.

I am very proud of my CSS Design and all the little animations and effects that improve the user-experience.

React Todo App

A Todo-App with User Authentication

This is a todo-app with user authentication and modern CSS Design. It is a MERN application (MongoDB, ExpressJS, ReactJS, NodeJs) that uses jsonwebtoken to authenticate the users (for login and signup). The users todos are stored in a Mongo database to be in sync across all devices.

The users can filter their todos with their completion status (all, active, completed). There is also a Darkmode-Switch which changes the theme from light to dark. The preffered color theme get's stored in LocalStorage so that it is the same on the next visit.

Zombie Game

A 2D Zombie Highscore Game in JavaScript p5

This is a simple game developed using p5.js during my Coding Bootcamp in March 2023. The purpose of coding this game was to apply newly acquired JavaScript fundamentals and object-oriented programming in a beginner-friendly, practical project.

The player's objective is to survive as long as possible in a zombie-infested world. There are different difficulty levels that control the speed and spawn-frequency of the zombies. If a zombie reaches you, you lose a heart. When you have no hearts left, the game is over.

Zombie Spiel

Ein 2D Zombie Highscore Spiel in JavaScript p5

Dies ist ein simples Browser Spiel, das ich während meines Coding Bootcamps im März 2023 mit JavaScript p5 entwickelt habe.
